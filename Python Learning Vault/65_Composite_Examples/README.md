# 65_Composite_Examples — โปรแกรมรวมประกอบองค์ความรู้

โฟลเดอร์นี้เก็บ **โปรแกรมที่ประกอบองค์ความรู้หลายอย่าง** — อธิบายให้เห็นว่าเมื่อเขียนโปรแกรมจริง ต้องนำ concept หลายตัวมาประกอบกันอย่างไร

---

## 📚 รายการโปรแกรมรวม

| โปรแกรม | Concept หลัก | ระดับ | ไฟล์ |
|---------|--------------|-------|------|
| อ่านไฟล์ → นับคำ → สร้าง report → เขียนไฟล์ | File + Dict + String + Loop | กลาง | [Composite_อ่านไฟล์นับคำสร้างรายงาน.md](./Composite_อ่านไฟล์นับคำสร้างรายงาน.md) |
| Parse log → สรุปสถิติ → เขียน JSON | File + Parsing + Dict + JSON + Function | กลาง–สูง | [Composite_Parse_Log_สรุปสถิติ.md](./Composite_Parse_Log_สรุปสถิติ.md) |

---

## 🧩 โครงสร้างแต่ละไฟล์

แต่ละโปรแกรมรวมจะมี:

1. **สิ่งที่โปรแกรมทำ** — สรุปสั้น ๆ
2. **โค้ดเต็ม** — พร้อมคอมเมนต์
3. **แผนภาพ Concept** — ตารางบอกว่าแต่ละส่วนใช้ concept จากโน้ตไหน
4. **อธิบายสั้น ๆ** — ทำไมต้องประกอบหลาย concept
5. **ลิงก์ที่เกี่ยวข้อง** — โน้ตหลัก + เกร็ดความรู้

## 📌 วิธีใช้

- **หลังเรียนขั้น 1–7** ใน [Learning_Path](../00_Central_Core/Learning_Path.md) แล้ว → มาดูโปรแกรมรวมเพื่อเห็นภาพรวม

- **เพิ่มโปรแกรมรวมใหม่** → ใช้โครงด้านบน + อัปเดต [Concept_Connections](../00_Central_Core/Concept_Connections.md)
