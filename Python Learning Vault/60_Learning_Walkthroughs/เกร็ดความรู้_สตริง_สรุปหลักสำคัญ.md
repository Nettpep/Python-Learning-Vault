## 🎯 สรุปสั้นๆ

สตริงไม่ใช่แค่ข้อความนิ่งๆ แต่เป็น **วัตถุ (Object)** ที่มี **Method** ช่วยจัดการเยอะมาก เพื่อให้เราไม่ต้องเขียน Loop ไปไล่เช็คทีละตัวอักษรเอง

---

## 1. การรวมสตริง (Concatenation)

การใช้เครื่องหมาย **`+`** คือการนำสตริงมาต่อกัน — **โดยไม่มีช่องว่างให้โดยอัตโนมัติ**

หากต้องการช่องว่าง ต้องใส่เพิ่มเอง เช่น `x + " " + y`

```python
x = "Hello"
y = "World"
print(x + y)       # "HelloWorld"
print(x + " " + y) # "Hello World"
```

---

## 2. การใช้ `in` เป็นตัวดำเนินการตรรกะ

ใช้ **`in`** เพื่อเช็คว่า **"มีข้อความนี้อยู่ในอีกข้อความหนึ่งหรือไม่"**

ผลลัพธ์ที่ได้จะเป็น **`True`** หรือ **`False`**

```python
'n' in 'banana'   # True
'nan' in 'banana' # True
'z' in 'banana'   # False
```

---

## 3. การเปรียบเทียบสตริง (Comparison)

- ใช้ **`==`** เช็คว่าเหมือนกันเป๊ะไหม
- ใช้ **`<`** หรือ **`>`** จะเปรียบเทียบตาม **ลำดับอักษร (Alphabetical Order)**

**ข้อควรระวัง:** Python มองว่า **ตัวพิมพ์ใหญ่ "เล็กกว่า" ตัวพิมพ์เล็ก** (เช่น `'Z'` มาก่อน `'a'` ในลำดับการเปรียบเทียบ)

```python
"apple" == "apple"  # True
"apple" < "banana" # True (เรียงตามตัวอักษร)
"Z" < "a"          # True (ตัวใหญ่ถือว่าเล็กกว่าตัวเล็ก!)
```

---

## 4. สตริงคือ "วัตถุ" (Object) และมี Method ในตัว

เนื่องจากสตริงเป็น Object มันจึงมีความสามารถ (Method) ติดตัวมาด้วยมากมาย เช่น:

| Method | ความหมาย |
|--------|----------|
| **`.lower()`** | ทำสำเนาเป็นตัวพิมพ์เล็ก (ตัวต้นฉบับไม่เปลี่ยน) |
| **`.upper()`** | ทำสำเนาเป็นตัวพิมพ์ใหญ่ |
| **`.find(คำ)`** | ค้นหาตำแหน่งของคำที่ต้องการ (ถ้าไม่เจอจะได้ `-1`) |
| **`.replace(old, new)`** | ค้นหาและแทนที่คำทั้งหมดที่เจอ |
| **`.strip()`** | ตัดช่องว่าง (รวมถึง Tab และ Newline) ที่หัวและท้ายสตริงออก |
| **`.startswith(คำ)`** | เช็คว่าสตริงเริ่มต้นด้วยคำที่ระบุหรือไม่ (คืนค่า True/False) |

```python
s = "  Hello World  "
s.lower()           # '  hello world  '
s.find("World")     # 6
s.replace("World", "Python")  # '  Hello Python  '
s.strip()           # 'Hello World'
s.startswith("  H") # True
```

## 5. การสแกนและดึงข้อมูล (Parsing and Slicing)

เทคนิคการดึงข้อมูลเฉพาะส่วน (เช่น **ดึงชื่อ Domain จากอีเมล**) โดยผสมผสาน:

1. หาจุดเริ่มต้นด้วย **`.find('@')`**
2. หาจุดสิ้นสุด (เช่น ช่องว่างถัดไป) ด้วย **`.find(' ', start_pos)`** หรือใช้ตำแหน่งที่เหมาะสม
3. ตัดเอาส่วนที่ต้องการด้วย **Slicing `[start : end]`**

```python
email = "user@example.com more text"
at_pos = email.find('@')           # 4
space_pos = email.find(' ', at_pos) # -1 ถ้าไม่มีช่องว่าง; ใช้ len(email) แทนได้
if space_pos == -1:
    space_pos = len(email)
domain = email[at_pos + 1 : space_pos]  # 'example.com'
```

## 6. ความแตกต่างระหว่าง Python 2 และ 3

**Python 3:** สตริงทุกตัวเป็น **Unicode โดยพื้นฐาน** ทำให้รองรับภาษาต่างๆ ทั่วโลกได้ง่ายและเป็นระบบกว่า Python 2 มาก
