# For Loop หาค่าที่น้อยที่สุด (ใช้ None เป็นตัวตั้งต้น)

**หมวด:** 20_Logic_Flow — Loops  
**วันที่บันทึก:** 2026-03-08  
**Tags:** `#learning` `#walkthrough` `#for-loop` `#list` `#min` `#None`

---

## 🎯 สิ่งที่เรียนรู้

โค้ดชุดนี้ใช้ **For Loop** เพื่อหา **ค่าที่น้อยที่สุด (Smallest)** จากรายการตัวเลข ความน่าสนใจคือการใช้ **`None`** เพื่อจัดการกับรอบแรก — เป็นเทคนิคที่โปรแกรมเมอร์นิยมใช้ เพราะทำให้ไม่ต้องเดาค่าเริ่มต้น (เช่น 0 หรือเลขใหญ่) และทำงานได้แม้ list เป็นตัวเลขลบทั้งหมด

---

## 📝 โค้ดตัวอย่าง

```python
smallest = None
print('Before')
for value in [9, 41, 12, 3, 74, 15]:
    if smallest is None:
        smallest = value
    elif value < smallest:
        smallest = value
    print(smallest, value)
print('After', smallest)  # 3
```

---

## 🔍 การเตรียมตัวแปร (Initial State)

- **`smallest = None`:** กำหนดให้ `smallest` เป็น "ความว่างเปล่า" เพราะยังไม่เริ่มดูตัวเลข จึงยังไม่รู้ว่าค่าไหนน้อยที่สุด
- **`print('Before')`:** แสดงข้อความว่ากำลังจะเริ่มเข้าสู่กระบวนการค้นหา

---

## 🔧 กลไกใน For Loop

โปรแกรมหยิบตัวเลขจาก `[9, 41, 12, 3, 74, 15]` มาทีละตัว และทำตามกฎนี้:

| เงื่อนไข | ความหมาย |
|----------|----------|
| **`if smallest is None`** | ใช้สำหรับ **รอบแรกสุด** เท่านั้น — เมื่อหยิบเลขตัวแรกมา (9) `smallest` ยังเป็น `None` จึงเก็บเลขนั้นลง `smallest` ทันที เป็นตัวตั้งต้นสำหรับเปรียบเทียบรอบถัดไป |
| **`elif value < smallest`** | ในรอบต่อๆ มา โปรแกรมถามว่า "เลขตัวใหม่ **น้อยกว่า** ค่าที่เล็กที่สุดที่เก็บไว้หรือเปล่า?" ถ้าน้อยกว่า → อัปเดต `smallest`; ถ้าไม่ → ไม่ทำอะไร |

---

## 🔍 Trace — จำลองเหตุการณ์ทีละรอบ

| รอบที่ | `value` (ตัวเลขที่หยิบมา) | การตัดสินใจของโปรแกรม | `smallest` (ค่าที่น้อยที่สุดในมือ) |
|--------|---------------------------|------------------------|-----------------------------------|
| เริ่มต้น | — | — | None |
| 1 | 9 | เข้า `if smallest is None` (รอบแรก) | 9 |
| 2 | 41 | 41 < 9 คือ เท็จ | 9 |
| 3 | 12 | 12 < 9 คือ เท็จ | 9 |
| 4 | 3 | 3 < 9 คือ จริง! (อัปเดตค่า) | 3 |
| 5 | 74 | 74 < 3 คือ เท็จ | 3 |
| 6 | 15 | 15 < 3 คือ เท็จ | 3 |

---

## 💡 บทสรุป

- **`print(smallest, value)`:** ในทุกรอบที่วนลูป โปรแกรมพิมพ์ค่าที่น้อยที่สุด ณ ขณะนั้นคู่กับตัวเลขที่กำลังดูอยู่
- **`print('After', smallest)`:** เมื่อดูครบทุกตัว โปรแกรมหลุดจากลูปและแสดงผลลัพธ์สุดท้าย — ได้ **3**

---

## 📌 ทำไมต้องใช้ `is None`?

การใช้ **`is None`** ช่วยป้องกันปัญหาเมื่อชุดตัวเลขเป็น **ลบทั้งหมด**:

- ถ้าตั้งต้น `smallest = 0` และ list คือ `[-5, -10]` โปรแกรมจะตอบว่า **0** (ผิด — ค่าที่น้อยที่สุดควรเป็น -10)
- การใช้ **`None`** บังคับให้โปรแกรมหยิบ **เลขตัวแรกในชุด** มาเป็นตัวตั้งต้นเสมอ จึงได้ผลลัพธ์ถูกต้องไม่ว่า list จะเป็นบวก ลบ หรือผสม
